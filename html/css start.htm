HTML(HyperText Markup Language) ជា​ភាសា​សម្រាប់បង្កើត​វេបផេក។ វា​អាច​ប្រើ​ដើម្បី ​សរសេរ​អត្ថបទ​ ធ្វើ​បញ្ជី​​តារាង​ ដាក់​រូបភាព​សម្លេង​វីដូអូ​​។ល។

CSS(Cascading Style Sheets) ជា​ភាសា​សម្រាប់​រចនា​វេបផេក​ឲ្យ​​ស្អាត។ វា​អាច​​រចនា​ ពុម្ពអក្សរ ពណ៌​អក្សរ ពណ៌​​ផ្ទៃ​​ខាង​​ក្រោយ គម្លាត ស៊ុម រឹម ទីតាំង គ្រោងវេបផេក ។ល។

អត្ថបទ

អត្ថបទ​តែង​មាន​ចំណង​ជើង​តាម​ឋានានុក្រម។ វា​​មាន​ច្រើន​វគ្គ​ច្រើន​ស្នើ​ទៅ​តាម​ទំហំ​របស់​ប្រធានបទ។ វគ្គ​អត្ថបទ​នីមួយៗ​ក៏​អាច​បែង​ចែក​ជា​ចំណែក​តូច​ៗជា​ អក្សរមួយតួ ពាក្យ​មួយឬច្រើន ឬ​មួយ​ប្រយោគ។ល។ វា​ក៏​មាន​ការ​សង្កត់​ន័យ​ឬ​សង្កត់​ធ្ងន់​ត្រង់​ចំណុច​សំខាន់​ៗ។ កន្លែងខ្លះនៃកឋាខណ្ឌក៏​ត្រូវ​សរសេរ​លើ​បន្ទាត់​គោល​ឬ​ក្រោម​បន្ទាត់​គោល​ទៀត​​ផង។

បញ្ជី

បញ្ជី​ធម្មតារាយ​ទិន្នន័យ​ពី​លើ​ចុះ​ក្រោម។ បញ្ជី​ខ្លះ​លំដាប់​លំដោយ​សំខាន់ រី​ឯ​បញ្ជី​ខ្លះ​លំដាប់​លំដោយ​មិន​សូវ​សំខាន់​នោះ​ទេ។​ ​មាន​ប្រភេទ​បញ្ជី​ដែល​អាច​រាយ​​នូវ​អ្វី​ជា​​គូរៗ​ផង​ដែរ។
